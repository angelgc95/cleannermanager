# Reviewer Access Notes

## App Access Summary

- This Android app is for invited cleaners only.
- Host accounts cannot use the Android app.
- Reviewers need a valid cleaner account that has already been invited by a host.

## What You Need To Create Before Submission

Create one dedicated reviewer cleaner account in production with:

- at least one assigned listing
- at least one upcoming cleaning event
- at least one checklist template already attached to that event
- at least one guide available

## Fill These Values Before Submitting

- Reviewer email: `REPLACE_WITH_REVIEWER_CLEANER_EMAIL`
- Reviewer password: `REPLACE_WITH_REVIEWER_PASSWORD`

## Reviewer Instructions To Paste In Play Console

`This app is a cleaner-only operational app for invited cleaning staff.

Use the reviewer credentials provided in this section to sign in.

After sign-in, review:
1. Dashboard
2. Calendar
3. Checklists
4. Shopping List
5. Settings

The test account has one assigned property and at least one scheduled cleaning event so the main workflow can be reviewed without extra setup.

Host accounts are intentionally not supported inside the Android app.`

## Submission Risk If This Is Missing

- Google review can be blocked or rejected if the reviewer cannot log in or cannot reach the main app experience.
